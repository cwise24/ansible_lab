Installation
====

MacOS - install `homebrew <https://howtogeek.com/211541/homebrew-for-os-x-easily-installs-desktop-apps-and-terminal-utilities/>`_
 - brew install python3
 - pip3 install ansible

Linux - install python3-pip (apt/yum)
 - pip3 install ansible
